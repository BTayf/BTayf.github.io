Integritetspolicy

1. Introduktion

Denna integritetspolicy gäller för Tayf ("appen"), som tillhandahålls av Tayf Barkawi ("vi", "oss"). Vi värnar om din integritet och vill förtydliga hur appen hanterar personuppgifter.

2. Insamling av data

Appen samlar inte in, lagrar eller delar några personuppgifter.

3. Tredje parter

Eftersom appen inte samlar in någon information, delas heller ingen data med tredje parter.

4. Dina rättigheter

Eftersom ingen data samlas in, behöver du inte vidta några åtgärder för att begära åtkomst, ändring eller radering av information.

5. Ändringar i integritetspolicyn

Om denna integritetspolicy uppdateras kommer ändringarna att publiceras här.
